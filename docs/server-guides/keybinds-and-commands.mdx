## Commands

Here you can find some useful commands for Venture Roleplay.

### Roleplay Commands

These commands can be used to facilitate your roleplay

| Command                     | Description                                  | Proximity/Global |
| --------------------------- | -------------------------------------------- | ---------------- |
| `/char` **OR** `/character` | Switch Character                             | N/A              |
| `/manager`                  | Open Character (Ped) Manager                 | N/A              |
| `/ooc`                      | Sends an out of character message            | Global           |
| `/do`                       | Sends an action message                      | Proximity        |
| `/me`                       | Sends a message in third person              | Proximity        |
| `/mer`                      | Same as `/me`, in red text                   | Proximity        |
| `/dob`                      | Displays your characters date of birth       | Proximity        |
| `/twitter`                  | Send a Twotter message in-game               | Global           |
| `/gme`                      | Send a message in third person               | Global           |
| `/darkweb`                  | Send an anonymous message                    | Global           |
| `/whoami`                   | Displays your characters information         | Private          |
| `/loadout`                  | Equip LEO Loadout                            | N/A              |
| `/call911`                  | Send a 911 call into dispatch & active units | CAD              |
| `/clear`                    | Clear all weapons                            | N/A              |

### Vehicle Commands

| Command       | Description                               |
| ------------- | ----------------------------------------- |
| `/eng`        | Toggle Engine                             |
| `/hood`       | Toggle Vehicle Hood                       |
| `/trunk`      | Toggle Vehicle Trunk                      |
| `/door [#]`   | Open/Close Door (Based on given number)   |
| `/window [#]` | Open/Close Window (Based on given number) |

### LEO Commands

| Command        | Description                                         |
| -------------- | --------------------------------------------------- |
| `/cuff`        | Cuff Nearest Player                                 |
| `/drag`        | Drag Nearest Player                                 |
| `/lidarweapon` | Equip Lidar Weapon                                  |
| `/lidar`       | Toggle Lidar Display                                |
| `/lidarwipe`   | Deletes all saved lidar data                        |
| `/lidarshow`   | Shows lidar display to nearest player for 5 seconds |
